### 2. Decode in C
```c
#include "../../src/c-api/crimsonkey_api.h"

uint8_t encrypted[] = { 0x94, 0xf7, 0x2b, 0x02, 0x0f, 0x0f, 0x0c };

if (crimsonkey_decrypt(encrypted, sizeof(encrypted), &decrypted, &length, 0) == CRIMSONKEY_SUCCESS) {
```
